
## Answers to Questions

### 1. What is the difference between getElementById, getElementsByClassName, and querySelector / querySelectorAll?

Ans/. getElementById return single element, getElementsByClassName returns collection of element, and querySelector / querySelectorAll use css selector return element.

### 2. How do you create and insert a new element into the DOM?

Ans/. document.createElement() to create a new element and use appendChild() to insert it into the DOM.

### 3. What is Event Bubbling? And how does it work?

Ans/. Even bubbling is a mechanism where as event triggerd on an element propagate up though ancestors inthe DOM tree.

### 4. What is Event Delegation in JavaScript? Why is it useful?

Ans/. Event delegation is a technique that leverages event bubbing to handle events for multiple elements with a single event listener

### 5. What is the difference between preventDefault() and stopPropagation() methods?

Ans/. preventDefault() stop the browser's default action for an event, while stopPropagation() stops the event from propagating further up the DOM tree.
